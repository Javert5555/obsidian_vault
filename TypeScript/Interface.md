Интерфейс - это над-объектная сущность, которая позволяет описать форму объекта, или то как он будет выглядеть в будущем.

_Интерфейс - это определение кастомного типа данных, но без реализации, он определяет свойства и методы, которые объект должен реализовать. Интерфейсы определяются с помощью ключевого слова **interface**._

```ts
interface User {
    name: string,
    age: number
}

const obj1: User = {
    name: 'name1',
    age: 33
}

console.log(obj1) // Object { name: "name1", age: 33 }
```

Правильно говорить, что obj1 реализует интерфейс User.
## Отличие Interface от Type

Type задаёт псевдоним для любой разновидности типов, включая примитивы

```ts
interface User {
    name: string,
    age: number
}

type newUser = {
    name: string,
    age: number
}
```
